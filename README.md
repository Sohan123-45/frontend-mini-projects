# 📦 Frontend Mini Projects

A collection of **small frontend projects** built while learning **HTML, CSS, and JavaScript**.  
These projects helped me strengthen my understanding of **core frontend concepts, DOM manipulation, and UI development**.

---

## 🚀 About This Repository

This repository contains multiple **mini projects** that I built during my frontend learning journey.

The main goal of these projects was to:

- Practice **HTML structure**
- Improve **CSS styling and layouts**
- Learn **JavaScript fundamentals**
- Understand **DOM manipulation**
- Build small **interactive UI components**

These projects are simple, beginner-friendly, and focused on improving practical frontend skills.

---

## 🛠️ Tech Stack

The projects in this repository are built using:

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**

No frameworks were used so that the focus remains on understanding the **core concepts of frontend development**.

---

## 📂 Project Structure

Each project is stored in its own folder.

Typical structure:

```
project-name/
│
├── index.html
├── style.css
└── script.js
```

This keeps every project **clean, organized, and easy to understand**.

---

## ▶️ How to Run the Projects

1. Clone this repository

```
git clone https://github.com/Sohan123-45/frontend-mini-projects.git
```

2. Navigate into the repository

```
cd frontend-mini-projects
```

3. Open any project folder.

4. Open the `index.html` file in your browser.
